# octocode-website

Site corporate d'Octocode SARL — `https://octocode-dev.com`

## Stack

- HTML / CSS / JS pur
- Hébergement GitHub Pages
- Polices Google Fonts (Bricolage Grotesque, Manrope, JetBrains Mono)

## Déploiement

1. Push ce repo sur GitHub (public ou private — GitHub Pages fonctionne sur les deux comptes payants, ou public en gratuit)
2. Repo → Settings → Pages → Source : `Deploy from a branch` → `main` / `/ (root)`
3. Custom domain : `octocode-dev.com` + cocher Enforce HTTPS
4. Configurer les DNS chez OVH (voir doc)

## Mise à jour du contenu

Édite simplement `index.html`, push sur main, GitHub Pages redéploie tout seul en quelques secondes.

## Mentions légales

Toutes les mentions SARL (SIRET, TVA, RCS, capital, siège, gérant) sont en pied de page, en FR et en EN. Mises à jour si la société change de capital ou de siège.
